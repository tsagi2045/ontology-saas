| 드래그 앤 드롭 CSV 업로드 | textarea 대신 파일 드래그 영역 + 열 매핑 프리뷰 UI | 중간 | ✅ 완료 |
| 반응형 레이아웃 | 사이드바 축소 모드, 모바일 대응 | 중간 | ✅ 완료 |

## 2. 그래프 시각화 고도화

| 아이디어 | 설명 | 난이도 | 상태 |
|---------|------|--------|------|
| 자동 레이아웃 알고리즘 | 현재 그리드 배치 → dagre/elk 기반 계층형/방사형 레이아웃 전환 | 중간 | ✅ 완료 |
| 노드 위치 저장 | 사용자가 드래그한 노드 위치를 DB에 persist | 낮음 | ✅ 완료 |
| 노드 그룹핑/클러스터 | 같은 클래스 엔티티를 박스로 그룹핑하여 시각적 구분 | 중간 | ✅ 완료 |
| 경로 하이라이트 | 두 노드 선택 시 최단 경로를 강조 표시 | 중간 | 🔲 |
| 노드 크기 변화 | 연결 관계가 많은 노드일수록 크게 표시 (노드 중요도 시각화) | 낮음 | 🔲 |
| 그래프 에니메이션 | 노드 추가/삭제 시 부드러운 전환 애니메이션 | 낮음 | 🔲 |
| 포커스 모드 | 특정 노드 클릭 시 해당 노드 중심으로 N-hop 이웃만 보여주는 뷰 | 중간 | 🔲 |
| 그래프 스냅샷 저장 | 현재 그래프 뷰를 PNG/SVG로 내보내기 | 중간 | 🔲 |
| 타임라인 슬라이더 | `createdAt` 기준으로 시간에 따른 그래프 성장 과정 재생 | 높음 | 🔲 |

## 3. 분석/인사이트 기능

| 아이디어 | 설명 | 난이도 | 상태 |
|---------|------|--------|------|
| 그래프 통계 대시보드 | 노드 차수(degree) 분포, 평균 연결 수, 클러스터링 계수, 허브 노드 식별 | 중간 | 🔲 |
| 중심성 분석 | PageRank, Betweenness Centrality 계산 → 핵심 엔티티 자동 식별 | 높음 | 🔲 |
| 고립 노드 탐지 | 관계가 없는 엔티티를 경고로 표시 | 낮음 | 🔲 |
| 중복 엔티티 탐지 | 이름/속성이 유사한 엔티티 자동 감지 → 병합 제안 | 중간 | 🔲 |
| 관계 패턴 분석 | "A→B→C" 같은 빈번한 관계 패턴을 자동 발견 | 높음 | 🔲 |
| 대시보드 차트 확장 | 관계 유형별 분포, 월별 엔티티 생성 추이, 클래스별 속성 완성도 | 중간 | 🔲 |

## 4. 추론 엔진 강화

| 아이디어 | 설명 | 난이도 | 상태 |
|---------|------|--------|------|
| OR 조건 지원 | 현재 AND만 가능 → AND/OR 중첩 조건 그룹 | 중간 | 🔲 |
| 체인 추론 | 규칙 A 결과가 규칙 B 트리거 → 연쇄 추론 실행 | 중간 | 🔲 |
| 규칙 스케줄링 | 매일/매시간 자동 실행 (cron 기반) | 중간 | 🔲 |
| 추론 결과 시각화 | 추론으로 생성된 관계를 그래프에서 점선+색으로 구분 | 낮음 | 🔲 |
| 규칙 충돌 감지 | 두 규칙이 같은 엔티티에 모순되는 액션을 하려는 경우 경고 | 높음 | 🔲 |
| 규칙 임포트/익스포트 | 규칙 세트를 JSON으로 공유 가능 | 낮음 | 🔲 |
| 규칙 실행 히스토리 | 언제 어떤 규칙이 실행되어 무엇이 변경되었는지 타임라인 | 중간 | 🔲 |
